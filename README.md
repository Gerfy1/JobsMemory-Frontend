# Jobs Memory - Frontend

## Sobre
Este repositório contém o projeto **frontend** da aplicação **Jobs Memory**, desenvolvido em **Angular 19** 
O frontend é uma **Single Page Application (SPA)** que consome a API RESTful do backend (java) para gerenciar candidaturas a vagas de emprego, acompanhar etapas de processos seletivos, definir lembretes e visualizar estatísticas

O objetivo deste projeto é oferecer uma interface intuitiva e responsiva para que o usuário gerencie suas candidaturas de forma organizada e eficiente.

## Funcionalidades
- Registro e login de usuários  
- Visualização e gerenciamento de candidaturas  
- Configuração de lembretes para entrevistas e follow-ups  
- Dashboard com estatísticas das candidaturas  

## Tecnologias Utilizadas
- Angular 19  
- TypeScript  
- HTML, CSS, SCSS  
- Consumo de API RESTful  
- Render & Vercel (deploy)

## Como Rodar Localmente
1. Clone este repositório:
```bash
git clone https://github.com/Gerfy1/JobsMemory-Front.git
```
2. Instale as dependências: 
```bash
npm install
```
3. Rode a aplicação:
```bash
ng serve
```
4. Acesse http://localhost:4200 no navegador
Certifique-se de que o backend do Jobs Memory esteja rodando para que a aplicação funcione corretamente.
